## How to Add Your Music

1. **Add your music file** to this folder (`public/music/`)
2. **Name it**: `birthday-song.mp3`
3. **Supported formats**: MP3, OGG, WAV (MP3 recommended)

## Recommended Music

- Happy birthday songs
- Romantic background music
- Upbeat celebration music
- Your favorite song together!

## File Size Tips

- Keep file size under 5MB for best performance
- Use MP3 format (smaller file size)
- You can compress audio files online if needed

## How It Works

- Click the **"ðŸ”‡ Music Off"** button in the top-right corner to start music
- Music will loop automatically
- Volume is set to 50% (you can adjust in code if needed)
- Music will auto-play when the tree forms (after you click play once)

## Changing the Music File

If you want to use a different filename:

1. Open `src/App.tsx`
2. Find line with: `src="/music/birthday-song.mp3"`
3. Change `birthday-song.mp3` to your filename
